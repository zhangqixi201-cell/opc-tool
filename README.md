# OPC Fork Protocol

一个可分叉、可追溯、可验证的一人公司经营实验协议。

## 核心闭环

`Fork → Run → Prove → Review → Repair → Save as Specimen → Fork Again`

## 功能

- Fork Lab：生成 7 天最小经营实验
- 系统质检：实验质量评分与自动修复建议
- 母本标本：三个深母本、适用边界与失败信号
- 裂变模拟器：人群、交付与资产变异建议
- 实验复盘：记录反馈、验证与失败假设
- 本地母本库：通过浏览器 `localStorage` 保存
- OPC-1.0：JSON 最小协议对象导出

## 本地运行

直接用浏览器打开 `index.html`。

## GitHub Pages

在仓库的 **Settings → Pages** 中，将部署来源设置为：

- Branch: `main`
- Folder: `/ (root)`

保存后即可获得公开网站地址。
